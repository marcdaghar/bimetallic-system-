# Bimetallic Monetary Systems and the Dynamics of Precious Metals

This repository contains the code for the paper "Bimetallic Monetary Systems and the Dynamics of Precious Metals: A Formal Model of the Yusuf-Grondona Nuqud System" by Marc Daghar.

## Overview

The framework implements:
1. Gold (dinar) and silver (dirham) stock dynamics
2. Yusufian production cycle
3. Basculement switching rule
4. Natural exchange rate R = (G/A)·(M_Au/M_Ag)
5. Optimal minting policy
